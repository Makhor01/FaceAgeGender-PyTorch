# Age & Gender Recognition - Local Real-Time

## Описание
Этот проект использует нейронную сеть на PyTorch для определения возраста и пола человека в реальном времени с веб-камеры. Модель обучена на датасете UTKFace.

- **Пол:** классификация Male/Female с вероятностью (confidence)  
- **Возраст:** регрессия, выводится как целое число  
- **Сглаживание:** для уменьшения прыжков используется скользящее среднее по последним N кадрам  

---

## Требования
- Python 3.9–3.12
- GPU не обязателен, но ускоряет работу

---

## Установка зависимостей
Можно установить все зависимости с помощью `pip`:

```bash
python setup_env.py
```
Или вручную:
```bash
pip install -r requirements.txt
```
---
## Файлы проекта

- run_camera.py – скрипт для запуска камеры и распознавания возраста и пола

- age_gender.pt – обученная модель PyTorch

- setup_env.py – скрипт для установки зависимостей

- requirements.txt – список необходимых библиотек
---
## Использование

- Подключите веб-камеру к компьютеру.

- Убедитесь, что файл модели age_gender.pt находится в той же папке, что и run_camera.py.

- Запустите:
```python
python run_camera.py
```
Нажмите Q, чтобы выйти из программы.
